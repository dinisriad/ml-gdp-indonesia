# Design Guide — GDP Prediction App

## Color Palette
- Primary: #1D6FA4 (biru)
- Secondary: #2ECC71 (hijau)
- Background: #F8F9FA
- Text: #212529
- Accent: #E74C3C (merah untuk peringatan)

## Typography
- Font utama: sans-serif default Streamlit
- Heading: bold, ukuran besar
- Body: reguler, ukuran 14-16px

## Komponen UI
- Sidebar: navigasi antar halaman
- Cards: tampilkan metrik/angka penting
- Expander: info tambahan yang bisa dibuka/tutup
- Metric card: tampilkan MAE, RMSE, R2

## Spacing
- Padding antar section: 1.5rem
- Gap antar kolom: 1rem
